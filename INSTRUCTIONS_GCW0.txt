
Compilation 
=============

rm gcw0build -rf
mkdir gcw0build
cd gcw0build
cmake -DCMAKE_TOOLCHAIN_FILE="/opt/gcw0-toolchain/usr/share/buildroot/toolchainfile.cmake" ..
make

If it fails to build (especially with static toolchain) then edit this file :
gcw0build/src/smw/CMakeFiles/smw.dir

And change it to :
